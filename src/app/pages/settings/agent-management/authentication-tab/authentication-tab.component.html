What Is Two Factor Authentication?

Two factor authentication (2FA) adds an extra layer of security to your account. Along with your username and password, you'll enter a unique verification code — sent by email, SMS, or both — before you can log in. Even if someone knows your password, they can't access your account without that code.

How Is Two Factor Authentication Helpful?

  1. Enhanced Security: Protects against unauthorized access, since users can only log in if they can receive the verification code sent to their email or phone.
  2. Peace of Mind: Ensures only authorized users can access their accounts from both the web and the mobile admin app.
  3. Anti-Phishing: Reduces the risk of phishing attacks.

What Happens When Two Factor Authentication Is Turned On?

  1. Login Process: After entering their username and password, users receive a unique code through their selected method — email, SMS, or both.
  2. Code Delivery: Every login attempt sends a new verification code to the selected email address, phone number, or both.
  3. Verification: Enter the code in the provided field to complete login.
  4. For enhanced security, all mobile app users must re-authorize once 2FA is enabled.
